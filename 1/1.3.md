# 1.3 値

## 1.3.1

`multiply (0,infinity)` は x == 0 なので x を簡約して 0 == 0 となる。このとき x を返すので、最終的な戻り値は 0 となる。

`multiply (inifinity,0)` は x == 0 となり x を簡約て innfinity が停止しない。そのためこの関数は停止しない。
